
# Conversor Alura Samuel.

Foi passado o desafio aos devs da turma do back-end, o desafio de criar um conversor de moedas e temperaturas.

Aonde é capaz converter 
                Reais a Euros
								Reais a Dólares
								Reais a Libras
								Reais a Yenes
								Reais a Won Coreano
								Dólares a Reais
								Euro a Reais
								Libra a Reais
                Celsius a Fahrenheit
